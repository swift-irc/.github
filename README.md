# swift-irc/.github

This repository carries the public GitHub organization-profile surfaces for
`swift-irc`.

## Purpose

- host the organization profile README shown on GitHub
- keep short, public-facing copy in one canonical repo
- stay intentionally small and easy to maintain

## Structure

- `.github/profile/README.md` - GitHub organization profile copy

## Canon

- keep this repo focused on the public org profile
- keep reusable visual assets in `swift-irc-brand`
- keep code and maintained forks in sibling `swift-irc` repos
